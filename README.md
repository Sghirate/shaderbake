# ShaderBake

Shader Compiler for cross compiling shaders for use with SDL3 GPU. Emits a header containing blob arrays for all provided input shaders.

## Dependencies

* SDL3. As CMake target `SDL3::SDL3`. If not provided, it will try to find a local version, or fetch SDL3 from GitHub.
* SDL_shadercross. As CMake target `SDL3_shadercross::SDL3_shadercross-static`. If not provided, it will fetch SDL_shadercross from GitHub.

## Usage Example
```bash
Usage: shaderbake [options] <input files>

Required options:
  -o | --output <value>            Output file. Will contain all generated shader blobs.

Optional options:
  -h | --help                      Show this help message and exit.
  -g | --debug                     Enable debug information in the generated shaders.
  -w | --workers <value>           Number of worker threads for parallel compilation. Defaults to logical CPU core count.
  -I | --include <directory>       Directory to search for include files.
  -D<name>[=<value>]               Define a preprocessor macro. Can be specified multiple times.

Input files should be of the format: [<dir>/]<name>(.vert|.frag|.comp)(.spv|.hlsl)
```

Command
```bash
/path/to/shaderbake -o /path/to/project/src/shaders.h -I /path/to/project/src/shaders /path/to/project/src/shaders/simple.vert.hlsl /path/to/project/src/shaders/simple.frag.hlsl
```
Resulting `/path/to/project/src/shaders.h`
```c
// File generated by ShaderBake. Manual changes are unlikely to persist!
#ifndef __GENERATED_SHADERS__
#define __GENERATED_SHADERS__
#include <stdint.h>

const uint8_t simple_vertex_SPIRV[...] = {
    ...
};
const uint8_t simple_vertex_DXIL[...] = {
    ...
};
const uint8_t simple_vertex_MSL[...] = {
    ...
};
const uint8_t simple_fragment_SPIRV[...] = {
    ...
};
const uint8_t simple_fragment_DXIL[...] = {
    ...
};
const uint8_t simple_fragment_MSL[...] = {
    ...
};
#endif // __GENERATED_SHADERS__

```

## Standalone Build

In order to use the tool by itself.

1. Clone this repository
```bash
git clone https://github.com/Sghirate/shaderbake && cd shaderbake
```

2. Compile
```bash
#Configure
cmake -B build -G Ninja
# Build
cmake --build build
```

3. Run
```bash
./bin/shaderbake <see Usage Example>
```

## CMake Integrated Build

* Using git submodules
* With a command line open in the project directory
* Assuming external tools & dependencies are stored in `./src/ext`
* SDL3 and SDL_shadercross being built/provided by the project is optional

1. Add shaderbake as a submodule
```bash
git submodule add https://github.com/Sghirate/shaderbake ./src/ext/shaderbake
```

2. Include shaderbake in the project; E.g. root `CMakeLists.txt`
```cmake
...
add_subdirectory(src/ext/shaderbake)
...
```

3. Add dependency task to project using shaders
```cmake
# ShaderBake Task. Will cross compile all shaders from shader_SOURCES into shaders.h
set(shader_SOURCES
  shaders/simple.frag.hlsl
  shaders/simple.vert.hlsl
  shaders/skybox.frag.hlsl
  shaders/skybox.vert.hlsl
)
add_custom_target(shaderbake_run
  COMMAND shaderbake -I ${CMAKE_CURRENT_SOURCE_DIR}/shaders -o ${CMAKE_CURRENT_SOURCE_DIR}/shaders.h ${shader_SOURCES}
  DEPENDS shaderbake
  WORKING_DIRECTORY ${CMAKE_CURRENT_SOURCE_DIR}
  COMMENT "run shaderbake in ${CMAKE_CURRENT_SOURCE_DIR}"
  SOURCES ${shader_SOURCES}
)
# App using baked shaders
set(app_SOURCES
  main.c
)
add_executable(app ${app_SOURCES})
target_link_libraries(app PRIVATE
  SDL3::SDL3
)
# Dependency. Ensures shaderbake runs before building the App
add_dependencies(app shaderbake_run)
```
